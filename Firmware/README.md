# Firmware
- Firmware developed in Verilog using AMD Vivado 2024.1
- Tenagra software and FPGA drivers from NumatoLab (FPGA board manufacturer) needed for bitstream upload (see NumatoLab website)
- Pregenerated bitstream file **bitstream_deepfifo_50kHz.bit** available for upload to board (sampling frequency 50kHz, ADC gain=4)
