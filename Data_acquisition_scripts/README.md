# Data acquisition scripts
These are example scripts for collecting data from the __NeuroADC__ device using Python and converting raw data to Matlab format.
- Script "**readout.py**" connects to the device and records data. Duration of recording and output file name can be edited in the script. Pay attention to using the serial number of your Mimas FPGA board for opening the connection.
- Script "**data_conversion.m**" reads the raw binary data file generated by the previous scripts and converts its to 32-channel format plus 8 separate trigger signals. Output is stored in a .mat file.
The scripts below are not needed for normal device operation.
- Script "**compute_CRC_lookup.m**" generates a lookup table used by data_conversion.m for CRC verification. It does not need to be run if its output "crctable.mat" is already present in the same folder.
- Script "**clean_chip.py**" resets the USB chip embedded in the FPGA board to its standard operating mode.
